##数组去重（字符串去重）

[...new Set(array)];//[1,2,3,4]

	deal.includes(item)?'':deal.push(item);
})

##生成二维数组
```
	getDyadicArray(array,count=4){
		let dyadic = [];
		let row = array.length/4;
		for(let i=0;i<row;i++){
			let start = i*4;
			let end = (i+1)*4;
			dyadic.push(array.slice(start,end));
		}
		return dyadic;
	}
```
